# 🎬 Telegram Movie Bot

A Telegram bot that allows users to search for movies, view ratings, and explore details using the TMDb API.

---

## 🚀 Features

- 🔍 Search movies using `/movie <name>`
- 🎯 Top results with interactive buttons
- ⭐ View ratings and movie details
- 🖼️ Displays movie posters
- ⚡ Fast async API requests using httpx

---

## 🛠️ Tech Stack

- Python
- python-telegram-bot
- TMDb API
- httpx
